# Когда использовать

Кнопка — элемент, который часто используется на веб-страницах. Её функция — это призыв к действию. Призыв должен быть
максимально конкретным и понятным, чтобы пользователь знал, что произойдёт после нажатия кнопки.

## Описание

Могут быть следующие виды кнопок:

- Button — обычная стандартная кнопка c текстом или с текстом и иконкой;
- Button Icon — кнопка с иконкой;
- Button Price — кнопка с текстом и ценой.

### Button

Представлена в 4-х размерах:

- 32
- 44
- 52
- 72

Могут быть следующие варианты кнопок:

- primary
- primary-alternative
- secondary
- ghost
- always-white
- blur
- negative

#### Button Icon

При клике на Button Icon вызывается действие в зависимости от кейса. В кнопке может использоваться только иконка.

Представлена в 4-х размерах:

- 24
- 32
- 44
- 52

Могут быть следующие варианты кнопок-иконок:

- primary
- primary-alternative
- secondary
- ghost
- always-white
- blur
- negative 
- scroll

Размер иконки не конролируется компонентом. Необходимо передавать иконку в зависимости от размера кнопки.

##### Button Price

Button Text Price используется в сценарии покупки товара. В правой части может быть расположена только цена.

Кнопки представлены в 3-х размерах:

- 44
- 52
- 72

Могут быть следующие варианты кнопок:

- primary
- primary-alternative
- always-white
- blur
